A data-centric framework for the cloud

A cloud-friendly framework built for synchronizing your applications over the Internet. Built to take advantage of the PHP ecosystem.

Build your data flows around APIs, message queues flat files or search engines to make them work together smoothly. With integrated reporting and logging, get insights on what is going on and keep control.

Middleware

It is an open-source project that allows you to create flow management services, describing them in a configuration format. Through YAML or JSON files, we can describe the flows, their sources, their destinations and the transformations to be applied to the data.
A compiler is responsible for transforming this configuration into executable and optimized PHP code.

Satellites

These services, once started, will be complementary to web applications or business applications already installed. They will improve their connectivity, without really part of these applications.
They will be like satellites, always nearby and complementary.
